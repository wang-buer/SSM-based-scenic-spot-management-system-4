## 1、项目介绍

基于SSM的景点管理系统拥有两种角色

管理员：景区管理、酒店管理、留言管理、用户管理等

用户：登录注册、景区酒店查看、发布回复帖子等


## 2、项目技术

后端框架：SSM（Spring、SpringMVC、Mybatis）

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven项目
- 硬件环境：Windows 或者 Mac OS
